# E-commerce Website

A full-featured E-commerce website developed with HTML, CSS, and JavaScript. This project serves as a modern online shop platform, providing users with product listings, a shopping cart, and a responsive design optimized for various devices.

## Features

- **Product Listings**: Browse a wide range of products with images, descriptions, and prices.
- **Shopping Cart**: Add, remove, or update quantities of products in a persistent cart.
- **Responsive Design**: Looks great and functions well on desktops, tablets, and smartphones.
- **Client-Side Interactivity**: JavaScript-driven navigation, item management, and UI behaviors.
- **User-Friendly Interface**: Clean layout using CSS for easy browsing and shopping.

## Tech Stack

- **HTML** (43.7%): Semantic structure for all website pages and sections.
- **JavaScript** (32.6%): Handles interactivity, cart logic, and UI dynamics.
- **CSS** (23.7%): Custom styles for layout, colors, and responsiveness.

## Getting Started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/ANANYA-BERA/E-commerce-Website.git
   cd E-commerce-Website
   ```

2. **Open in Browser**
   - Open `index.html` in your favorite web browser.
   - Explore all available pages and features.

3. **Customize**
   - Update `products` data, images, or styles as needed.
